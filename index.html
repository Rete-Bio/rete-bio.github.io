The Architecture

Neuro-Symbolic by Design.

LLMs (System 1) are creative but unreliable. Rete adds a deterministic Reasoning Engine (System 2) to enforce biological constraints.

01

Generative Proposal

An LLM acts as the "creative search" engine, scanning literature and multimodal data to propose novel hypotheses or trial designs.

→
02

Logic Compiler

The Rete Engine translates the proposal into executable code. It checks the code against our Axiomatic Knowledge Graph for logical violations.

→
03

Secure Execution

The validated code runs in a sandboxed environment against your raw data. If the code errors, the hypothesis is rejected. Zero hallucinations.
